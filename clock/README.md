# Clock

## 背景介绍

时钟。

### 功能介绍

墙上时钟

- 按键`Esc`退出GUI
- 按钮`switch`切换秒针是否连续转动，初始时不连续转动

数字时钟

- 按键`Esc`退出GUI
- 按钮`24/12`切换小时制，初始时为24小时制
